Design and Develop Custom Application (Mini Project) using Salesforce Cloud

Aim
To design and develop a custom application in Salesforce Cloud for managing student records.

Step 1: Login to Salesforce
1. Open Salesforce Developer Edition.
2. Login using your Salesforce account.

Step 2: Create a Lightning Application
1. Go to:
   Setup → App Manager
2. Click:
   New Lightning App
3. Enter:
   App Name: Student Management System
4. Click:
   Next → Next → Next → Save & Finish

Step 3: Create Custom Object
1. Go to:
   Setup → Object Manager
2. Click:
   Create → Custom Object
3. Enter:
   Label: Student
   Plural Label: Students
   Record Name: Student Name
4. Click Save

Step 4: Create Custom Fields
Go to:
Student → Fields & Relationships → New

Create the following fields:
1. Number → Roll_no
2. Text → Course
3. Email → Email

Click:
Next → Next → Save
for each field.

Step 5: Create Student Tab
1. Go to:
   Setup → Tabs
2. Under:
   Custom Object Tabs
   click New
3. Select Object:
   Student
4. Choose any tab style.
5. Click:
   Next → Next → Save

Step 6: Add Tab to Application
1. Go to:
   Setup → App Manager
2. Find:
   Student Management System
3. Click:
   ▼ → Edit
4. Open:
   Navigation Items
5. Move:
   Student
   to Selected Items.
6. Click:
   Next → Save

Step 7: Open Application
1. Click App Launcher (9 dots icon).
2. Search:
   Student Management System
3. Open the application.

Step 8: Add Student Records
1. Open:
   Students
2. Click:
   New
3. Enter sample data:
   Student Name: Prasad
   Roll_no: 101
   Course: Computer Engineering
   Email: prasad@gmail.com
4. Click Save

Result
The custom Salesforce Cloud application for managing student records was successfully created and executed.

Conclusion
The mini project was successfully implemented using Salesforce Cloud by creating a custom Lightning App, custom object, custom fields, and record management functionality.
